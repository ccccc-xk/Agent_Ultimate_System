## 检索流程详细

```mermaid
sequenceDiagram
    participant U as 用户
    participant API as FastAPI
    participant ES as Elasticsearch
    participant MV as Milvus
    participant RR as Reranker
    participant CP as Compressor
    participant LLM as Qwen-Plus

    U->>API: POST /api/rag/query
    par 并发双路检索
        API->>ES: BM25 关键词检索
        ES-->>API: ~20 条候选 + BM25 分数
        API->>MV: 向量语义检索
        MV-->>API: ~20 条候选 + 余弦相似度
    end
    API->>API: RRF 融合排序
    API->>RR: 重排 ~20 条候选
    RR-->>API: Top-3 (score >= 0.6)
    API->>CP: 压缩 Context
    CP-->>API: 压缩后文本 + 压缩报告
    API->>LLM: 生成回答
    LLM-->>API: 回答文本
    API-->>U: {answer, sources, confidence, tokens_used}
```
