## 1. 企业知识库问答

### POST `/api/rag/query` — 企业知识库问答

输入自然语言问题，通过 ES + Milvus 混合检索 + Reranker 重排序，从企业文档中召回相关内容，用 LLM 生成回答。

**请求体**：

| 参数 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| question | string | 是 | — | 自然语言问题 |
| top_k | int | 否 | 3 | 最终返回结果数（1-10） |

**curl**：
```bash
curl -X POST http://localhost:9005/api/rag/query \
  -H "Content-Type: application/json" \
  -d '{"question":"员工年假有几天？","top_k":3}'
```

**响应**：
```json
{
  "answer": "根据《公司规章制度手册》，员工年假规定如下：\n\n1. 工作满1年不满10年的，年休假**5天**\n2. 工作满10年不满20年的，年休假**10天**\n3. 工作满20年的，年休假**15天**\n\n国家法定休假日、休息日不计入年假假期。",
  "sources": [
    {
      "doc_name": "公司规章制度手册.md",
      "chunk_text": "第四章 休假管理 第十五条 员工年假天数...",
      "score": 0.92
    }
  ],
  "confidence": 0.89,
  "tokens_used": 512,
  "elapsed_ms": 2300
}
```

## 2. 健康检查

### GET `/api/rag/health` — 服务健康状态

检查所有依赖服务（ES、Milvus、Reranker、Ollama）的连接状态。

**curl**：
```bash
curl http://localhost:9005/api/rag/health
```

**响应**：
```json
{
  "status": "healthy",
  "services": [
    {"name": "elasticsearch", "status": "ok", "detail": "http://localhost:9200"},
    {"name": "milvus", "status": "ok", "detail": "localhost:19530"},
    {"name": "reranker", "status": "ok", "detail": "bge-reranker-large"},
    {"name": "ollama", "status": "ok", "detail": "http://localhost:11434"}
  ]
}
```

状态值：`healthy`（全部正常）/ `degraded`（部分降级）/ `unhealthy`（不可用）

## 技术架构

```
用户提问
  ↓
ES 全文检索 (BM25) ──┐
                      ├→ RRF 融合 → Reranker 重排序 → Top-K 文档片段
Milvus 向量检索 ──────┘
  ↓
Ollama (qwen2.5:1.5b) 生成回答
  ↓
返回: answer + sources + confidence
```
