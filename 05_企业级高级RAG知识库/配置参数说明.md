# 配置参数说明

所有配置通过 `.env` 文件或环境变量设置。

## LLM 配置

| 参数 | 默认值 | 说明 |
|------|--------|------|
| OPENAI_API_KEY | - | OpenAI 兼容 API Key（DashScope） |
| OPENAI_BASE_URL | `https://dashscope.aliyuncs.com/compatible-mode/v1` | API 端点 |
| LLM_MODEL_NAME | `qwen-plus` | 模型名称 |

## Milvus 配置

| 参数 | 默认值 | 说明 |
|------|--------|------|
| MILVUS_HOST | `localhost` | Milvus 服务地址 |
| MILVUS_PORT | `19530` | Milvus 服务端口 |
| MILVUS_COLLECTION | `enterprise_knowledge` | Collection 名称 |

## Elasticsearch 配置

| 参数 | 默认值 | 说明 |
|------|--------|------|
| ES_HOST | `http://localhost:9200` | ES 服务地址 |
| ES_INDEX | `enterprise_knowledge` | 索引名称 |

## Embedding 配置

| 参数 | 默认值 | 说明 |
|------|--------|------|
| EMBEDDING_MODEL_PATH | `BAAI/bge-large-zh-v1.5` | Embedding 模型路径或 HuggingFace ID |
| EMBEDDING_DIMENSION | `1024` | 向量维度 |

## Reranker 配置

| 参数 | 默认值 | 说明 |
|------|--------|------|
| RERANKER_URL | `http://localhost:9997` | XInference 服务地址 |
| RERANKER_MODEL | `bge-reranker-large` | Reranker 模型名称 |
| RERANKER_THRESHOLD | `0.6` | 最低重排分数阈值 |

## RAG 配置

| 参数 | 默认值 | 说明 |
|------|--------|------|
| TOP_K | `3` | 最终返回的文档数量 |
| BM25_WEIGHT | `0.4` | BM25 检索在 RRF 融合中的权重 |
| VECTOR_WEIGHT | `0.6` | 向量检索在 RRF 融合中的权重 |
| RRF_K | `60` | RRF 公式中的常数 k |
| SIMILARITY_THRESHOLD | `0.5` | 最低相似度阈值 |
| CHUNK_SIZE | `500` | 文本分块大小（字符数） |
| CHUNK_OVERLAP | `50` | 文本分块重叠大小（字符数） |
| HYBRID_TOP_K | `20` | 单路检索返回的候选数量 |

## LangSmith 配置

| 参数 | 默认值 | 说明 |
|------|--------|------|
| LANGCHAIN_TRACING_V2 | `true` | 启用 LangChain Tracing |
| LANGCHAIN_API_KEY | - | LangSmith API Key |
| LANGCHAIN_PROJECT | `enterprise-rag` | LangSmith 项目名 |

## API 配置

| 参数 | 默认值 | 说明 |
|------|--------|------|
| API_PORT | `8002` | API 服务端口 |
