# 常用药物说明

## 一、阿莫西林胶囊

### 基本信息
- **通用名称**：阿莫西林胶囊
- **商品名**：阿莫仙、再林、阿莫西林克拉维酸钾（复方制剂）
- **药物分类**：青霉素类抗生素
- **医保类别**：甲类

### 适应症
- 上呼吸道感染：扁桃体炎、咽炎、鼻窦炎、中耳炎
- 下呼吸道感染：急性支气管炎、肺炎
- 泌尿生殖系统感染：尿道炎、膀胱炎、肾盂肾炎
- 皮肤软组织感染：蜂窝织炎、伤口感染
- 幽门螺杆菌感染（联合用药）

### 用法用量
- **成人**：口服，每次0.5g（2粒），每6-8小时一次，一日3次
- **儿童**：按体重计算，每日25-50mg/kg，分3次服用
- **疗程**：一般5-7天，严重感染可延长至10-14天
- **服用时间**：饭后服用可减少胃肠道不适
- **注意事项**：用温水送服，不可掰开或咀嚼缓释制剂

### 禁忌症
- 对青霉素类药物过敏者禁用
- 有青霉素过敏性休克史者绝对禁用
- 传染性单核细胞增多症患者禁用（易发生皮疹）

### 不良反应
- **常见**（发生率>1%）：
  - 胃肠道反应：恶心、呕吐、腹泻、食欲不振
  - 皮疹（发生率约5%-10%）
- **少见**：
  - 转氨酶升高（一过性）
  - 嗜酸性粒细胞增多
- **罕见但严重**：
  - 过敏性休克（需立即抢救）
  - 伪膜性肠炎
  - Stevens-Johnson综合征

### 药物相互作用
- 与丙磺舒合用：可延长阿莫西林半衰期
- 与甲氨蝶呤合用：可增加甲氨蝶呤毒性
- 与口服避孕药合用：可能降低避孕效果
- 与华法林合用：可能增强抗凝作用

### 特殊人群用药
- **孕妇**：B类药物，权衡利弊后可使用
- **哺乳期**：少量分泌入乳汁，需权衡利弊
- **老年患者**：肾功能减退者需调整剂量
- **肝肾功能不全**：严重肾功能不全需延长给药间隔

## 二、布洛芬

### 基本信息
- **通用名称**：布洛芬
- **商品名**：芬必得、美林（混悬液）、布洛芬缓释胶囊
- **药物分类**：非甾体抗炎药（NSAIDs）
- **医保类别**：甲类

### 适应症
- 发热（体温≥38.5℃时的退热治疗）
- 轻至中度疼痛：头痛、牙痛、痛经、关节痛、肌肉痛、神经痛
- 骨关节炎、类风湿关节炎的症状缓解
- 术后疼痛

### 用法用量
- **成人退热**：每次0.2-0.4g，每4-6小时一次，24小时不超过1.2g
- **成人镇痛**：每次0.2-0.4g，每4-6小时一次
- **缓释制剂**：每次0.3g，每12小时一次
- **儿童退热**：每次5-10mg/kg，每6-8小时一次，24小时不超过4次
- **服用时间**：建议饭后服用，减少胃肠刺激

### 禁忌症
- 对布洛芬或其他NSAIDs过敏者禁用
- 活动性消化性溃疡或出血者禁用
- 严重肝肾功能不全者禁用
- 严重心力衰竭患者禁用
- 妊娠晚期（≥28周）禁用
- 冠状动脉搭桥手术围手术期禁用

### 不良反应
- **常见**：
  - 胃肠道：恶心、呕吐、胃痛、消化不良
  - 神经系统：头晕、头痛
- **少见**：
  - 皮疹、瘙痒
  - 水肿、血压升高
- **罕见但严重**：
  - 消化道出血、穿孔
  - 严重皮肤反应（Stevens-Johnson综合征）
  - 肝肾功能损害
  - 心血管事件（长期大剂量使用风险增加）

### 药物相互作用
- 与阿司匹林合用：增加消化道出血风险
- 与抗凝药（华法林）合用：增强抗凝作用，出血风险增加
- 与利尿剂合用：可能减弱降压效果
- 与其他NSAIDs合用：不良反应叠加，禁止同时使用
- 与锂盐合用：可升高血锂浓度

### 特殊人群用药
- **孕妇**：前6个月权衡利弊，妊娠晚期禁用
- **哺乳期**：少量分泌入乳汁，短期使用相对安全
- **儿童**：6个月以上可使用，严格按体重计算剂量
- **老年人**：消化道出血风险增加，建议使用最低有效剂量

## 三、二甲双胍

### 基本信息
- **通用名称**：盐酸二甲双胍
- **商品名**：格华止、盐酸二甲双胍片、盐酸二甲双胍缓释片
- **药物分类**：双胍类降糖药
- **医保类别**：甲类

### 适应症
- 2型糖尿病的一线治疗药物
- 联合其他降糖药治疗血糖控制不佳的患者
- 肥胖或超重的2型糖尿病患者（首选）
- 多囊卵巢综合征（辅助用药）

### 用法用量
- **普通片剂**：起始剂量每次0.25g，每日2-3次，餐中或餐后服用
- **逐渐加量**：根据血糖控制情况，每周增加0.25-0.5g
- **维持剂量**：每日1.0-1.5g，分2-3次服用
- **最大剂量**：每日不超过2.0g
- **缓释制剂**：每日一次，晚餐时服用，起始0.5g，最大可调至2.0g

### 禁忌症
- 肾功能不全（eGFR<30 mL/min/1.73m²）禁用
- 需要使用碘化造影剂检查前后48小时暂停使用
- 急性或慢性代谢性酸中毒（包括糖尿病酮症酸中毒）禁用
- 严重肝功能不全禁用
- 急性心力衰竭、休克禁用
- 酗酒者慎用（增加乳酸酸中毒风险）

### 不良反应
- **常见**（尤其在治疗初期）：
  - 胃肠道反应：恶心、呕吐、腹胀、腹泻、腹痛、食欲下降
  - 金属味
  - 以上症状多在用药初期出现，多数患者可在1-2周内逐渐耐受
- **少见**：
  - 维生素B12吸收减少（长期使用需监测）
  - 低血糖（单独使用很少发生，与胰岛素或磺脲类合用时可能）
- **罕见但严重**：
  - 乳酸酸中毒（发生率极低，但病死率高）

### 药物相互作用
- 与碘化造影剂合用：增加乳酸酸中毒风险，需暂停二甲双胍
- 与酒精合用：增加乳酸酸中毒风险
- 与糖皮质激素合用：可能升高血糖，需调整剂量
- 与噻嗪类利尿剂合用：可能影响血糖控制
- 与西咪替丁合用：可增加二甲双胍血药浓度

### 特殊人群用药
- **孕妇**：国内说明书列为禁用，国际指南认为可作为妊娠糖尿病的替代用药
- **哺乳期**：可少量分泌入乳汁，一般建议暂停哺乳或停药
- **老年人**：定期监测肾功能，根据eGFR调整剂量
- **儿童**：10岁以上2型糖尿病患儿可使用

### 用药注意事项
- 建议餐中或餐后服用以减少胃肠道不适
- 用药期间避免过量饮酒
- 定期监测血糖、糖化血红蛋白、肾功能
- 如需进行碘造影剂检查，提前48小时停药，检查后48小时复查肾功能后再恢复用药
