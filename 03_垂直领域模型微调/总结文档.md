# 项目三：垂直领域模型微调 — 总结文档

## 功能概述

原计划通过 SFT（Supervised Fine-Tuning）微调大语言模型，让模型学会"守规矩"输出标准 JSON 格式的业务数据。

**因本机 GPU 不足（RTX 3060 6GB），训练已取消。**

## 当前替代方案

| 需求 | 替代方案 | 说明 |
|------|---------|------|
| Agent 大脑（意图识别+推理） | DeepSeek deepseek-v4-flash | Dify 内置配置，通过 API 调用 |
| RAG 文档生成（P04/P05） | Ollama qwen2.5:1.5b | 本地运行，免费无限制 |

## 项目页面地址

| 服务 | 地址 | 说明 |
|------|------|------|
| Ollama | [http://localhost:11434](http://localhost:11434) | 本地 LLM 服务 |
| DeepSeek | [https://platform.deepseek.com](https://platform.deepseek.com) | 在线 API 控制台 |

> 本项目无独立 Web 页面，LLM 能力通过 P04/P05/P06 间接使用。

## 训练数据保留

训练数据和脚本已保留在 `data/cleaned/train_final.json`（41 条），未来有 GPU 可随时重训。

## 未来有 GPU 时的训练方案

```bash
# 1. 上传项目到 GPU 服务器（如 AutoDL RTX 4090）
scp -r 03_垂直领域模型微调/ root@<服务器IP>:/root/

# 2. 安装依赖并训练
cd /root/03_垂直领域模型微调
pip install -r requirements.txt
bash train.sh

# 3. 部署为 vLLM API
vllm serve output/checkpoint-xxx --port 9003 --api-key vertical-sft-secret-key

# 4. 在 Dify 中切换模型
# 设置 → 模型供应商 → 添加 OpenAI Compatible
# API Base: http://host.docker.internal:9003/v1
# 模型名称: vertical-sft-model
```
