## 如果未来想训练模型

### 1. 租 GPU 服务器

- 平台：AutoDL（autodl.com）
- GPU：RTX 4090（24GB 显存），约 2 元/小时
- 系统：Ubuntu 22.04 + CUDA 12.1

### 2. 上传项目并训练

```bash
# 上传到服务器
scp -r 03_垂直领域模型微调/ root@<服务器IP>:/root/

# SSH 登录后
cd /root/03_垂直领域模型微调
pip install -r requirements.txt
bash train.sh
```

### 3. 部署为 vLLM API

```bash
vllm serve output/checkpoint-xxx --port 8001 --api-key vertical-sft-secret-key
```

### 4. 在 Dify 中切换模型

- 设置 → 模型供应商 → 添加 OpenAI Compatible
- API Base: `http://host.docker.internal:8001/v1`
- 切换 Agent 模型到微调模型
