# 项目二：政企物流智能化转化 — 接口文档

> 后端地址：[http://localhost:9002](http://localhost:9002) | 技术栈：Spring Boot 3 + MyBatis-Plus + MySQL 8.x + WebSocket

## 1. NLP 智能派单 `/api/nlp`

### POST `/api/nlp/dispatch` — 口语化智能派单

接收口语化文本，LLM 自动提取地点、客户名、问题描述等信息，创建工单。

**请求体**：

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| text | string | 是 | 口语化报修/投诉描述 |

**curl**：
```bash
curl -X POST http://localhost:9002/api/nlp/dispatch \
  -H "Content-Type: application/json" \
  -d '{"text":"3号楼202房间电风扇坏了，一直嗡嗡响"}'
```

**响应**：
```json
{
  "code": 200,
  "msg": "工单创建成功",
  "data": {
    "id": 16,
    "location": "3号楼202房间",
    "clientName": null,
    "issue": "电风扇坏了，一直嗡嗡响",
    "actionsTaken": [],
    "assignedTo": null,
    "priority": "NORMAL",
    "status": "PENDING",
    "createdAt": "2026-06-11T20:33:00"
  }
}
```

---

### POST `/api/nlp/query` — 大白话查数据库

用自然语言提问，LLM 自动转为 SQL 查询并执行。

**请求体**：

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| question | string | 是 | 自然语言问题 |

**curl**：
```bash
curl -X POST http://localhost:9002/api/nlp/query \
  -H "Content-Type: application/json" \
  -d '{"question":"查询所有待处理的工单"}'
```

**响应**：
```json
{
  "code": 200,
  "msg": "查询成功",
  "data": {
    "sql": "SELECT * FROM t_work_order WHERE status = 'PENDING'",
    "columns": ["id", "location", "issue", "status", "priority"],
    "rows": [
      {"id": 16, "location": "3号楼202房间", "issue": "电风扇坏了", "status": "PENDING", "priority": "NORMAL"}
    ],
    "row_count": 1
  }
}
```

---

### GET `/api/nlp/query/history` — 查询历史

**参数**：`limit`（int，默认 20）

```bash
curl "http://localhost:9002/api/nlp/query/history?limit=10"
```

---

### DELETE `/api/nlp/query/history/{id}` — 删除查询历史

```bash
curl -X DELETE http://localhost:9002/api/nlp/query/history/1
```

---

### PUT `/api/nlp/query/history/{id}` — 编辑查询历史

```bash
curl -X PUT http://localhost:9002/api/nlp/query/history/1 \
  -H "Content-Type: application/json" \
  -d '{"question":"修改后的问题"}'
```

## 2. 工单管理 `/api/work-order`

### GET `/api/work-order/list` — 工单分页列表

**参数**：

| 参数 | 类型 | 说明 |
|------|------|------|
| page | int | 页码，默认 1 |
| size | int | 每页条数，默认 10 |
| status | string | 状态筛选：PENDING/IN_PROGRESS/COMPLETED |
| priority | string | 优先级筛选：HIGH/NORMAL/LOW |

**curl**：
```bash
curl "http://localhost:9002/api/work-order/list?page=1&size=10&status=PENDING"
```

**响应**：
```json
{
  "code": 200,
  "data": {
    "records": [
      {
        "id": 16,
        "location": "3号楼202房间",
        "clientName": null,
        "issue": "电风扇坏了",
        "actionsTaken": [],
        "assignedTo": null,
        "priority": "NORMAL",
        "status": "PENDING",
        "createdAt": "2026-06-11T20:33:00"
      }
    ],
    "total": 1,
    "pages": 1,
    "current": 1,
    "size": 10
  }
}
```

---

### GET `/api/work-order/{id}` — 工单详情

```bash
curl http://localhost:9002/api/work-order/16
```

---

### PUT `/api/work-order/status` — 更新工单状态

**请求体**：

| 参数 | 类型 | 说明 |
|------|------|------|
| id | long | 工单 ID |
| status | string | 新状态：PENDING/IN_PROGRESS/COMPLETED |

```bash
curl -X PUT http://localhost:9002/api/work-order/status \
  -H "Content-Type: application/json" \
  -d '{"id":16,"status":"IN_PROGRESS"}'
```

---

### PUT `/api/work-order` — 更新工单详情

```bash
curl -X PUT http://localhost:9002/api/work-order \
  -H "Content-Type: application/json" \
  -d '{"id":16,"location":"3号楼202房间","issue":"电风扇异响","assignedTo":"张师傅","priority":"HIGH"}'
```

---

### DELETE `/api/work-order/{id}` — 删除工单

```bash
curl -X DELETE http://localhost:9002/api/work-order/16
```
