## 意图路由流程

```mermaid
flowchart TD
    Start([用户输入]) --> Intent{意图分类}
    
    Intent -->|tool_query| ParamCheck{参数是否齐全?}
    Intent -->|tool_dispatch| Dispatch[调用 smart_dispatch]
    Intent -->|knowledge| RAG[调用 rag_knowledge_query<br/>企业知识库 :8083]
    Intent -->|medical| Medical[调用 medical_chat_query<br/>医疗知识库 :8082]
    Intent -->|human_handoff| Handoff[转人工话术]
    Intent -->|无法识别| Fallback[兜底回复]
    
    ParamCheck -->|缺少月份| AskMonth[追问: 请问哪个月份?]
    ParamCheck -->|缺少单号| AskOrder[追问: 请问订单号是多少?]
    ParamCheck -->|缺少关键词| AskKeyword[追问: 请提供查询关键词]
    ParamCheck -->|参数齐全| CallTool[调用对应工具]
    
    CallTool --> Format[格式化为自然语言回答]
    Dispatch --> Format
    RAG --> Format
    Medical --> Format
    
    Format --> Answer([输出回答])
    Handoff --> Answer
    Fallback --> Answer
    AskMonth --> Answer
    AskOrder --> Answer
    AskKeyword --> Answer
```

## 数据流图

```mermaid
sequenceDiagram
    participant U as 用户
    participant W as 聊天组件
    participant D as Dify Chatflow
    participant L as LLM (vLLM)
    participant T as 工具
    participant S as 后端服务

    U->>W: 输入消息
    W->>D: POST /v1/chat-messages (SSE)
    D->>L: 意图分类
    L-->>D: {intent: tool_query}
    D->>L: 参数检查
    L-->>D: {status: ready, tool: finance_report_query}
    D->>T: 调用工具
    T->>S: GET /api/finance/page?month=2026-03
    S-->>T: {data: {records: [...]}}
    T-->>D: 工具返回结果
    D->>L: 格式化回答
    L-->>D: 自然语言回答
    D-->>W: SSE 逐字推送
    W-->>U: 打字机效果显示
```
