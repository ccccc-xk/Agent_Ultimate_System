聊天组件嵌入演示

此页面演示如何在任意网页中一行代码嵌入 Agent 智能客服聊天组件。

使用说明： 替换下方 apiKey 为你在 Dify 中创建的 Chatflow 应用的 API Key。

嵌入方式

<!-- 1. 引入 SDK -->
<script src="chat-widget.js"></script>

<!-- 2. 初始化 -->
<script>
  AgentChatWidget.init({
    apiBase: 'http://your-dify-host:80',
    apiKey: 'app-your-api-key',
    primaryColor: '#1890ff',  // 可选：自定义主题色
    logo: '🤖'               // 可选：自定义 Logo
  });
</script>

API 方法

AgentChatWidget.open()         // 打开聊天窗口
AgentChatWidget.close()        // 关闭聊天窗口
AgentChatWidget.toggle()       // 切换开关
AgentChatWidget.clearHistory() // 清除对话历史
AgentChatWidget.destroy()      // 销毁组件
