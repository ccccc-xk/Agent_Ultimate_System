# 项目六：Agent 智能体合流总管 — 总结文档

## 功能概述

压轴大合流项目。通过 Dify 平台将前五个项目串联为一个智能体闭环：

**大白话输入 → 意图识别 → 自动调用工具/查知识库 → 自然语言回答**

核心能力：
- **意图识别**：自动判断用户意图，分流到不同处理路由
- **工具调用**：自动调用 P01 财务/用户查询、P02 智能派单/查库
- **知识库问答**：调用 P04 医疗知识库、P05 企业知识库
- **多轮对话**：支持上下文记忆，参数不足时主动追问
- **人工转接**：识别投诉/人工诉求时自动转人工
- **流式输出**：SSE 打字机效果，实时显示回答

## Agent 工作流程

```
用户输入
  ↓
[LLM 意图分类]
  ├─ tool_query → 检查参数 → 调用 P01/P02 工具 → 格式化回答
  ├─ tool_dispatch → 调用 P02 智能派单 → 返回工单信息
  ├─ knowledge → 调用 P04/P05 知识库 → 基于文档回答
  ├─ human_handoff → 转人工话术
  └─ default → 兜底回复
```

- 工具调用超时设置为 60 秒
