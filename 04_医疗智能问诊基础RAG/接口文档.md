## 1. 医疗问答

### POST `/api/chat` — 医疗知识库问答

输入医疗健康问题，从 Milvus 向量库检索相关文档片段，用 LLM 生成专业回答。

**请求体**：

| 参数 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| question | string | 是 | — | 医疗健康问题 |
| top_k | int | 否 | 3 | 检索文档片段数量（1-10） |

**curl**：
```bash
curl -X POST http://localhost:9004/api/chat \
  -H "Content-Type: application/json" \
  -d '{"question":"感冒了吃什么药？","top_k":3}'
```

**响应**：
```json
{
  "answer": "### 感冒了怎么办？\n\n#### 1. 一般治疗：\n- 多饮水、注意休息、保持室内通风。\n\n#### 2. 对症治疗：\n- 发热/头痛：对乙酰氨基酚或布洛芬缓解症状。\n- 鼻塞：使用伪麻黄碱滴鼻液减轻鼻塞。\n\n**重要提醒**：如果症状持续超过7天，请及时就医。",
  "sources": [
    {
      "doc_name": "常见疾病诊疗指南.md",
      "chunk_text": "感冒的治疗以对症处理为主...",
      "page_num": 1,
      "score": 0.87
    }
  ],
  "confidence": 0.85
}
```

### POST `/api/chat/stream` — SSE 流式医疗问答

与 `/api/chat` 相同功能，但以 SSE 流式返回，适合前端打字机效果展示。

**请求体**：同 `/api/chat`

**curl**：
```bash
curl -X POST http://localhost:9004/api/chat/stream \
  -H "Content-Type: application/json" \
  -d '{"question":"高血压怎么控制？"}' \
  --no-buffer
```

**响应格式**（SSE）：
```
event: sources
data: [{"doc_name":"常用药物说明.md","chunk_text":"...","page_num":1,"score":0.82}]

event: chunk
data: 高血压的控制主要包括以下几个方面：

event: chunk
data: 1. 生活方式调整：低盐饮食...

event: done
data: {"confidence":0.83}
```

## 2. 知识库管理 `/api/knowledge`

### POST `/api/knowledge/upload` — 上传医疗文档

上传 Markdown/PDF 文档到知识库，自动分块并建立向量索引。

**curl**：
```bash
curl -X POST http://localhost:9004/api/knowledge/upload \
  -F "file=@新药说明.md"
```

**响应**：
```json
{
  "doc_name": "新药说明.md",
  "chunk_count": 12,
  "elapsed_seconds": 3.5
}
```

---

### GET `/api/knowledge/list` — 列出已导入文档

```bash
curl http://localhost:9004/api/knowledge/list
```

**响应**：
```json
{
  "documents": [
    {"doc_name": "常见疾病诊疗指南.md", "chunk_count": 45, "created_at": "2026-06-10"},
    {"doc_name": "常用药物说明.md", "chunk_count": 38, "created_at": "2026-06-10"},
    {"doc_name": "医院科室介绍.md", "chunk_count": 22, "created_at": "2026-06-10"}
  ]
}
```

---

### DELETE `/api/knowledge/{doc_name}` — 删除文档

```bash
curl -X DELETE http://localhost:9004/api/knowledge/常见疾病诊疗指南.md
```

**响应**：
```json
{
  "deleted": true,
  "doc_name": "常见疾病诊疗指南.md",
  "deleted_chunks": 45
}
```
