# 项目一：企业管理基座系统 — 接口文档

> 后端地址：[http://localhost:9001](http://localhost:9001) | 技术栈：Spring Boot 3.2 + MyBatis-Plus + MySQL 8.x + JWT

---

## 认证说明

除登录接口外，所有接口需要在请求头中携带 JWT Token：

```
Authorization: Bearer <token>
```

获取 Token：
```bash
curl -X POST http://localhost:9001/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username":"admin","password":"123456"}'
```

响应：
```json
{
  "code": 200,
  "msg": "success",
  "data": {
    "id": 1,
    "username": "admin",
    "realName": "系统管理员",
    "phone": "13800000001",
    "roles": ["eyJhbGciOiJIUzM4NCJ9..."],
    "permissions": ["system:user:view", "finance:view", ...]
  }
}
```

Token 有效期 24 小时，过期需重新登录。

## 2. 仪表盘 `/api/dashboard`

## 3. 财务模块 `/api/finance`

### GET `/api/finance/page` — 财务分页查询

**参数**：

| 参数 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| pageNum | int | 否 | 1 | 页码 |
| pageSize | int | 否 | 10 | 每页条数 |
| month | string | 否 | — | 月份，如 "3月"、"2024年3月" |

**curl**：
```bash
curl "http://localhost:9001/api/finance/page?pageNum=1&pageSize=10&month=3%E6%9C%88" \
  -H "Authorization: Bearer <token>"
```

**响应**：
```json
{
  "code": 200,
  "data": {
    "records": [
      {
        "id": 1,
        "month": "3月",
        "income": 500000.00,
        "expense": 320000.00,
        "profit": 180000.00,
        "category": "主营业务",
        "remark": "3月份财务数据"
      }
    ],
    "total": 15,
    "pages": 2,
    "current": 1,
    "size": 10
  }
}
```

---

### POST `/api/finance/ai-summary` — AI 财务摘要

**参数**：`month`（query，string）— 月份

**curl**：
```bash
curl -X POST "http://localhost:9001/api/finance/ai-summary?month=3%E6%9C%88" \
  -H "Authorization: Bearer <token>"
```

**响应**：
```json
{
  "code": 200,
  "data": "3月份企业财务概况：总收入50万元，总支出32万元，净利润18万元。利润率为36%，较上月增长5%。主要支出项目为人力成本和办公费用..."
}
```

---

### POST `/api/finance` — 新增财务记录

**请求体**：
```json
{
  "month": "4月",
  "income": 600000.00,
  "expense": 350000.00,
  "profit": 250000.00,
  "category": "主营业务",
  "remark": "4月份数据"
}
```

**curl**：
```bash
curl -X POST http://localhost:9001/api/finance \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer <token>" \
  -d '{"month":"4月","income":600000,"expense":350000,"profit":250000,"category":"主营业务"}'
```

---

### PUT `/api/finance` — 更新财务记录

请求体同新增，需包含 `id` 字段。

---

### DELETE `/api/finance/{id}` — 删除财务记录

```bash
curl -X DELETE http://localhost:9001/api/finance/1 \
  -H "Authorization: Bearer <token>"
```

---

### POST `/api/finance/import` — Excel 批量导入

```bash
curl -X POST http://localhost:9001/api/finance/import \
  -H "Authorization: Bearer <token>" \
  -F "file=@财务数据.xlsx"
```

---

### GET `/api/finance/template` — 下载导入模板

```bash
curl -O http://localhost:9001/api/finance/template \
  -H "Authorization: Bearer <token>"
```

## 4. 用户模块 `/api/user`

### GET `/api/user/page` — 用户分页搜索

**参数**：

| 参数 | 类型 | 说明 |
|------|------|------|
| keyword | string | 搜索关键词（姓名/手机/邮箱） |
| pageNum | int | 页码，默认 1 |
| pageSize | int | 每页条数，默认 10 |

**curl**：
```bash
curl "http://localhost:9001/api/user/page?keyword=%E5%BC%A0&pageNum=1&pageSize=10" \
  -H "Authorization: Bearer <token>"
```

**响应**：
```json
{
  "code": 200,
  "data": {
    "records": [
      {
        "id": 1,
        "username": "zhangsan",
        "realName": "张三",
        "phone": "13800000002",
        "status": 1
      }
    ],
    "total": 1,
    "pages": 1
  }
}
```

---
